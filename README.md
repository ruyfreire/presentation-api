# presentation-api

API para consumir dados de apresentação pessoal

## Instalação

```bash
pnpm install
```

## Compilação e execução

```bash
# modo de desenvolvimento
pnpm run start

# modo de desenvolvimento watch
pnpm run start:dev

# modo de produção
pnpm run start:prod
```

## Testes

```bash
# testes unitários
pnpm run test

# testes e2e
pnpm run test:e2e

# cobertura de testes
pnpm run test:cov
```
